Built and deployed MausamOG, a multi-page GenAI-powered monsoon preparedness and citizen assistance platform. The deployed version includes a personalized readiness dashboard, AI-generated monsoon preparedness plans, emergency checklist tracking, localized alert cards, Mapbox risk visualization, travel advisory generation, multilingual safety assistant, and local emergency resource discovery. It now includes i18next UI language support for English, Hindi, Kannada, and Tamil, with a persistent switcher, cookie/localStorage persistence, localized form labels, and localized Server Action status/validation messages. The app includes magic-link login/signup, persisted plans and checklist state, cached/rate-limited assistant interactions, and deterministic fallbacks so the demo remains usable when external AI, email, or map services are unavailable. Built with Next.js App Router, Server Actions, TypeScript, Zod, Neon Postgres, Drizzle ORM, Upstash Redis, Gmail OAuth, and modular repository/service layers.
